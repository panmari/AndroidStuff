ApkBlueInstaller
----------------
(C) www.aplu.ch

Version 1.40

Transfer and install APKs from PC to Android Smartphone

Unpack zip and copy .jar AND lib subdirectory in any filder.
Click on ApkBlueInstaller.jar or create a link to it.

Under Windows you may execute ApkBlueInstaller.exe.

You need the Java Virtual Machine (JVM) installed on your Win/Mac/Linux machine.


To run the project from the command line, type the following:

java -jar ApkBlueInstaller.jar

----------------------------------------------

The Smartphone server app DroidInstall must be running
